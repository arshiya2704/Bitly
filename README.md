# Bitly_Prototype

Bitly Prototype using technologies:

• Frontend: ejs </br>
• Backend:  node.js </br>
• Database: MongoDB</br>

Features: 

• Encode </br>
• Decode</br>
• Trending Server to show top 10 hits</br>

Steps to run the application:

1. npm install
2. npm start
